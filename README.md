# Basic Network Sniffer

This project was completed as part of the CodeAlpha Cyber Security Internship.

## Description
A Python-based network sniffer that captures live network packets and displays
source IP, destination IP, protocol type, packet length, and timestamp.

## Tools & Technologies
- Python 3
- Scapy
- Kali Linux

## Features
- Real-time packet capture
- Protocol identification (TCP, UDP, ICMP)
- Timestamped packet analysis
- Lightweight and efficient

## How to Run
1. Install dependencies:
   sudo apt install python3-scapy
2. Run the script as root:
   sudo python3 network_sniffer.py

## Output
The tool displays live packet details in the terminal.
Screenshots are included in the screenshots folder.
